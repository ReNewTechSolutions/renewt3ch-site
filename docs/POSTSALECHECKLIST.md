# Post-sale checklist (Seller)

## Confirm with buyer
- [ ] Best contact email/channel
- [ ] Preferred handoff method:
  - [ ] ZIP only
  - [ ] GitHub repo transfer/share
- [ ] Hosting method:
  - [ ] buyer redeploys (recommended)
  - [ ] Vercel project transfer
- [ ] Domain transfer timing (IONOS)

## Deliver
- [ ] Send `renewt3ch-delivery.zip`
- [ ] Point buyer to: `docs/README.md`
- [ ] Provide demo URL (optional)
- [ ] Provide domain/DNS docs:
  - `DOMAIN-HANDOFF_IONOS.md`
  - `IONOS-DNS-RECORDS.md`

## Hosting (if included)
- [ ] Assist buyer deploy (or transfer Vercel project)
- [ ] Confirm custom domain added
- [ ] Confirm SSL is active

## Domain (IONOS)
- [ ] Complete push/transfer
- [ ] Buyer confirms ownership
- [ ] Buyer confirms DNS control

## Close out
- [ ] Buyer confirms site is live
- [ ] Buyer confirms OG preview works
- [ ] Offer optional services (if desired):
  - `SUPPORT-SCOPE.md`
  - `MAINTENANCE-PRICING.md`
