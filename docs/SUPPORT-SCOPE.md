# Support scope (Optional Services Addendum)

Base purchase includes the delivered files + documentation. Ongoing services are optional and only provided if purchased separately.

## Included with base delivery
- Delivery ZIP with full site + assets + docs
- Written deployment instructions
- Domain handoff guidance docs

## Optional services (examples)
### Admin / content updates
- Update text, links, CTAs
- Swap logos/images
- Minor layout adjustments within template system

### Hosting support (Vercel)
- Deploy + configure custom domain
- Assist with DNS cutover (IONOS)
- Verify SSL + OG previews

### Webflow rebuild (only if requested)
- Recreate the template in Webflow
- Map classes/components
- Configure Webflow forms/hosting/CMS (as scoped)

## Not included unless explicitly agreed
- Backend/API development
- Copywriting strategy or naming research
- Custom illustration/advanced motion work
- SEO campaigns/link building
- Unlimited revisions

## Access requirements
Buyer provides access for any requested support:
- Vercel (if used)
- Domain registrar (IONOS)
- Webflow (if used)

## Communication + expectations
- Primary channel: email (or agreed channel)
- Scope is confirmed before work begins
- Rush requests may require a rush fee
