1. Brand Overview

Brand name: renewt3ch
A compact, tech-forward brand identity designed for a web studio / AI-powered services business.

Positioning: “A futuristic, yet approachable partner for digital products, web experiences, and AI-powered tools.”

Core attributes
  • Futuristic, not cold: glowy gradients + clean geometry.
  • Technical, but friendly: readable type and simple layouts.
  • Modular: logos, icons, and palette work across web, print, and product mockups.
Core logo files
  • Logo-Renewt3ch.svg — gradient logotype
  • wordmark-black.svg — black wordmark
  • wordmark-pure-white.svg — white wordmark
  • r3-icon-gradient.svg — square gradient icon
  • r3-icon-black.svg — solid black icon
  • r3-icon-white.svg — solid white icon
Keep all master assets in /brand or /assets/identity.

2. Logo & Wordmark Usage

Primary logo

Use Logo-Renewt3ch.svg as the default logo on dark or gradient backgrounds. The wordmark is outlined and spaced for crisp rendering at a wide range of sizes.

Minimum recommended width for the full logo on screen: 200 px. For smaller UI elements, switch to the icon.

Monochrome wordmarks
  • wordmark-pure-white.svg — for dark, photo, or gradient backgrounds.
  • wordmark-black.svg — for light or near-white backgrounds.

These match the gradient logo’s geometry, including open counters and consistent stroke weights.

4. Typography

Primary Typeface
Poppins — Bold

Used for logos, page titles, major section headings, and key UI labels.

• H1 / H2 / H3
• Logo lockups (outlined in SVG assets)
• Navigation labels & key callouts

Body & UI Typeface
Inter — Regular

Body text and UI elements should use the Inter UI stack for clarity and performance.

Inter, system-ui, -apple-system, "Segoe UI", Roboto, sans-serif

Type scale (suggested)
  • H1: 2.2–2.6rem · 700 · tight letter-spacing
  • H2: 1.4–1.6rem · 700
  • H3: 1.1–1.25rem · 600
  • Body: 0.95–1rem · 400–500
  • Labels / UI: 0.75–0.85rem · 500 · uppercase where needed

8. Voice & Tone

Brand voice
  • Confident, not arrogant: explain the tech, don’t overhype it.
  • Clear & human: short sentences, concrete benefits, no jargon walls.
  • Curious & experimental: highlight prototypes, experiments, and labs.
Sample microcopy
  • “Spin up a launch-ready site in days, not months.”
  • “AI-powered where it matters. Human-led where it counts.”
  • “Switch on a cleaner, faster web experience for your users.”
